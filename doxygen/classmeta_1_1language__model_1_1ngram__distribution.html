Detailed Description

template<size_t N>
class meta::language_model::ngram_distribution< N >

Represents a smoothed distribution of ngrams of either words, POS tags, function words, or characters.

Smoothing is done with absolute discounting with the (n-1)-gram model recursively to the unigram level.

Calculates a smoothed probability distribution over ngrams.

\(P_{AD}(word|prev) = \frac{ max(c(prevword) - D, 0) }{ c(prev) } + \frac{D}{c(prev)} \cdot |S_w| \cdot P_{AD}(word)\)

template<size_t N>
void meta::language_model::ngram_distribution< N >::calc_discount_factor ( )
private

Calculate D = n1 / (n1 + 2 * n2).

n1 = number of ngrams that appear exactly once. n2 = number of ngrams that appear exactly twice.
