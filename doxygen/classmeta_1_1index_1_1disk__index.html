Detailed Description

template<class DerivedIndex>
class meta::index::disk_index< DerivedIndex >

Contains functionality common to inverted_index and forward_index; mainly, creating chunks and merging them together and storing various mappings.

Holds generic data structures and functions that inverted_index and forward_index both use.
