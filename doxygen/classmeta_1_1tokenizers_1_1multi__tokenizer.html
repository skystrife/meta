Detailed Description

The multi_tokenizer class contains more than one tokenizer.

This is useful for trying combined feature methods.

For example, you could tokenize based on ngrams of words and parse tree rewrite rules. The multi_tokenizer keeps track of all the features in one set for however many internal tokenizers it contains.
